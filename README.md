# Projects Repository

This repository collects academic and research-oriented projects developed during the Master's Degree in [**Computer Science**](https://www.unimi.it/en/education/master-programme/computer-science-master) at the University of Milan (Università degli Studi di Milano).

The projects span multiple areas of Artificial Intelligence, Machine Learning, Signal Processing, Deep Learning, Natural Language Processing, and Computational Neuroscience.

The main objective of this repository is to document practical implementations, experiments, and analyses carried out throughout different university courses.

---

# Repository Structure

## 1. [Decision Tree Classifier](./BinaryTreeClassifier)
### Statistical Methods for Machine Learning — Tree Predictors for Binary Classification

Implementation of a **Decision Tree classifier from scratch** for binary classification tasks.

## 2. [Recipe Generator with T5](./Recipe_Generator)
### Hurry up, I'm hungry!

Natural Language Processing project focused on **automatic recipe generation** using a fine-tuned **T5 Transformer** model.

## 3. [ECG Signal Processing](./BiomedicalSignalProcessing)
### ECG Signal Decomposition using Fourier Analysis

Implementation of an ECG signal decomposition framework based on the **Discrete Fourier Series (DFS)**.

## 4. [Melanoma Classification with CNNs](./DeepLearning)
### Binary Classification of Melanoma using CNNs

Development and evaluation of multiple **Convolutional Neural Networks (CNNs)** for melanoma classification.

## 5. [Interoceptive Active Inference](./Interoceptive Active Inference)
### Interoceptive Active Inference to Simulate Cardiac–Perceptual Coupling

Implementation of an **Active Inference** model inspired by the work of Micah Allen and Karl Friston.
